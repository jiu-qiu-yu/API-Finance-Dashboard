## ADDED Requirements

### Requirement: 全站界面中文化
系统 SHALL 将所有 UI 元素的文字替换为中文，包括但不限于：窗口标题、按钮文字、标签、提示信息、对话框内容、进度条文字、状态标识。

#### Scenario: 主窗口标题和按钮中文化
- **WHEN** 用户启动应用
- **THEN** 窗口标题显示为 "New API 站长助手"
- **AND** 主按钮显示为 "开始巡检"
- **AND** 设置按钮显示为 "设置"

#### Scenario: 巡检进度中文化
- **WHEN** 巡检进行中，正在检查站点 "ProviderX"（第 2 个，共 5 个）
- **THEN** 进度条文字显示 "正在巡检 ProviderX... (2/5)"

#### Scenario: 错误提示中文化
- **WHEN** 巡检出错
- **THEN** 弹窗标题为 "巡检错误"，内容为中文错误描述

#### Scenario: 设置面板中文化
- **WHEN** 用户打开设置面板
- **THEN** 所有标签、输入提示、按钮均为中文（如 "站点名称"、"站点类型"、"告警阈值" 等）
